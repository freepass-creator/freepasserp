높이 조절
손오공렌터카 주식회사약관 레이아웃 테스트

제1조 테스트 조항

1. 이 문단은 약관 2단 레이아웃이 A4 페이지에 좌측 → 우측 순서로 채워지는지 테스트하기 위한 더미 텍스트입니다.

2. column-fill: auto 설정 시 왼쪽 컬럼이 먼저 지정된 높이까지 채워진 후 오른쪽 컬럼으로 넘어갑니다.

3. 높이(height)를 적절히 설정하면 A4 경계선 안에 딱 맞게 들어가야 합니다.

제2조 추가 테스트

1. 담보의 종류 : 보증금 · 선납금 · 보증보험증권 · 신용.

2. 계약 체결 후 5영업일 이내 담보를 제공하여야 합니다.

3. 계약 정상 종료 시 보증금·보증보험증권은 반환합니다.

4. 선납금은 월 대여료에서 분할 차감합니다.

5. 계약 종료 시 미이행 채무를 공제한 후 반환합니다.

6. 잔존 채무가 있으면 회사는 고객에게 청구할 수 있습니다.

7. 차량 인수 시 인수가액에서 보증금을 상계합니다.

8. 보증·담보 제공을 이유로 채무 이행을 거절할 수 없습니다.

제3조 대여료

1. 대여료는 월 단위 정산합니다.

2. 납부 지연 시 연 15% 이내의 연체이자를 청구합니다.

3. 1개월 미만은 일 단위 환산합니다.

4. 대여료 변동 시 서면 합의합니다.

5. 과태료·통행료는 고객 부담입니다.

6. 차량 미반환 시 단기 대여 요금 적용합니다.

7. 탁송비는 고객 부담입니다.

제4조 운전자

1. 운전자 연령 이상 면허 소지자만 운전 가능합니다.

2. 운전자 범위에 해당하는 자만 운전할 수 있습니다.

3. 미등록 운전자 사고 시 보험 적용 제외됩니다.

제5조 차량 인도

1. 회사는 양호한 상태의 차량을 인도합니다.

2. 인수증에 서명 후 운행합니다.

3. 옵션 오기는 자동차등록증 기준입니다.

4. 하자 발견 시 즉시 기재합니다.

5. 차량 교체 요구 불가합니다.

6. 반납 시 원상 수리합니다.

7. 구조 변경 시 원상 복원합니다.

8. 연료는 인도 시와 동일하게 반납합니다.

9. 만료 30일 전 통보합니다.

10. 차량 내 물품은 폐기됩니다.

제6조 보험

1. 영업용 자동차보험 가입 차량을 대여합니다.

2. 보장 범위 내에서 보상합니다. 고의·금지행위·음주 등은 제외됩니다.

3. 자차 면책제도 — 면책금 납부 시 회사 부담합니다. 12대 중과실 시 20% 추가 부담합니다.

4. 사고 발생 시 즉시 신고하고 블랙박스 영상을 제공합니다.

5. 면책금 미납 시 보험 미처리됩니다.

6. 보험 조건 변경은 7일 전 협의합니다.
